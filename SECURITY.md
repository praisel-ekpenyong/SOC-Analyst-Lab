# Security Policy

## Supported Versions

We only support the most recent version of this project. Older versions may not receive security updates.

## Reporting a Vulnerability

We encourage responsible disclosure and appreciate your efforts to improve the security of this project.

If you discover a security issue or vulnerability, please report it privately using GitHub's "Report a vulnerability" button on the repository's Security tab. This creates a private discussion channel with the maintainers.

If you are unable to use the vulnerability reporting workflow, please send an email to ekpenyongpraisel@gmail.com with as much detail as possible (a description of the issue, how to reproduce it, and any other relevant information).

We will acknowledge your report within 48 hours and work with you to investigate and resolve the issue.

Thank you for helping keep this project secure.
