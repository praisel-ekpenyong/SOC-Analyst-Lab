# Screenshots Placeholder

This folder contains screenshots collected as evidence for each lab and ticket.

## How to Collect Screenshots

1. **Windows:** Press `Win + Shift + S` to capture a region, or `Print Screen` for full screen. Save as PNG.
2. **macOS:** Press `Cmd + Shift + 4` to capture a region.
3. **VirtualBox:** Use Machine → Take Screenshot from the VM menu.

## Naming Convention

Use this format: `labXX-step-description.png`
