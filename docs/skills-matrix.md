## Competency Summary by Role Type

### Tier 1 Help Desk
- Password resets, account unlocks, basic connectivity diagnosis
- Ticket creation, routing, and closure
- User communication and expectation management
- *Demonstrated by: All 12 tickets, Lab 04, Lab 01*

### Tier 2 Desktop Support
- Active Directory administration
- DNS and network troubleshooting
- Windows repair (SFC, DISM, Event Viewer)
- PowerShell automation
- *Demonstrated by: Lab 01, Lab 02, Lab 03, Lab 05, Incidents 1–4*

### System Administrator (Junior)
- Active Directory OUs, GPO, file shares
- LAMP server deployment (osTicket)
- PowerShell scripting
- Incident documentation and root cause analysis
- *Demonstrated by: Lab 01, Lab 04, Lab 05, All Incidents*
